Ваша задача - составить функцию, которая на входе получает уравнение в виде строки с цифрами, затертыми местами (‘#’) и одной из трёх арифметических операций (+, - или *), например - "##*##=302#". В результате необходимо вернуть цифру (из интервала 0-9), при подставлении которой вместо всех #, уравнение было бы верным. Или -1, если это сделать невозможно.

Важное примечание - на месте # не может быть ни одна из цифр, которые уже фигурируют в уравнении. Также, если после знака равенства (=) стоит 2 и более затертых символа, ответ не может состоять из нулей (00, 000 и т.д.).
Числа в формате #n, #nn и так далее, где n - любая цифра, не могут начинаться с нуля. То есть, "#9+3=12" == -1.
В случае, если несколько цифр подходят - используйте наименьшую из них. В качестве чисел, фигурирующих в формуле, могут быть как положительные (например, "1+1=#"), так и отрицательные, перед которыми стоит знак "-" ("19--45=5#", "-1*-6=#")

Входные данные: Шифр.

Выходные данные: Цифра кода.

Пример:

safeCode("-5#*-1=5#") == 0
safeCode("##*##=302#") == 5
safeCode("19--45=5#") == -1
safeCode("#9+3=12") == -1
{% else %} 
safe_code("-5#*-1=5#") == 0
safe_code("##*##=302#") == 5
safe_code("19--45=5#") == -1
safe_code("#9+3=12") == -1
{% endif %}

Как это используется: В криптографии для защиты ценной информации.

Предусловия:
ответ - [0-9] или -1
